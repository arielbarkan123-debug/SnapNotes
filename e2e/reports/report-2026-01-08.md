### File Upload Limitation
MCP browser tools cannot directly upload local files to HTML file inputs. This is a testing infrastructure limitation, not an app bug. Workarounds:
- Use Playwright MCP tools for file uploads
- Manual testing for upload scenarios
- Mock file uploads in unit tests
