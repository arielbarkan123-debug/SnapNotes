# Homework First-Message Upgrade — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** When a student clicks "Get Help", the AI immediately generates a full solution explanation (not a template), optionally with a diagram and step-by-step walkthrough button.

**Architecture:** Remove the template greeting from session creation. The session page auto-fires a chat request on mount when the conversation is empty. A new "Full Explanation" mode in the tutor engine generates a complete walkthrough instead of Socratic hints. A new "Include Diagram" toggle on the help form persists `enable_diagrams` in the session record.

**Tech Stack:** Next.js 14 (App Router), Supabase, Tailwind CSS, next-intl, Anthropic Claude API

**Branch:** `feat/homework-first-message-upgrade`

**Prerequisite:** Merge `feat/step-by-step-diagram-buildup` into `main` first.

```typescript
// Auto-trigger first AI explanation when session has empty conversation
const hasAutoStarted = useRef(false)

useEffect(() => {
  if (
    helpSession &&
    helpSession.conversation.length === 0 &&
    !isChatLoading &&
    !hasAutoStarted.current
  ) {
    hasAutoStarted.current = true
    // Send the auto-start sentinel — tutor engine will generate full explanation
    handleSendMessage('__auto_start__')
  }
}, [helpSession, isChatLoading, handleSendMessage])
```

**Step 3: Build a full-explanation system prompt**

Add a new constant `FULL_EXPLANATION_SYSTEM_ADDITION` after `SOCRATIC_TUTOR_SYSTEM_BASE` (around line 500):

```typescript
const FULL_EXPLANATION_SYSTEM_ADDITION = `

IMPORTANT MODE OVERRIDE — FULL EXPLANATION:
The student has just submitted their question. Instead of asking Socratic questions, provide a COMPLETE, CLEAR solution walkthrough:

1. Start with a brief "Let's solve this!" introduction (1 sentence)
2. Break the solution into numbered steps
3. For each step: show the math/reasoning clearly, explain WHY this step works
4. End with the final answer clearly stated
5. Invite follow-up questions: "Ask me if anything isn't clear!"

Do NOT ask the student questions first. Do NOT withhold the answer. Give them the full picture so they can learn from seeing the complete solution path.

Keep it concise but thorough. Use clear formatting with numbered steps.
`
```

**Step 4: Use the override when auto-start is detected**
